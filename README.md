# FoodLoop AI

FoodLoop AI is an AI-powered platform to reduce food waste by connecting restaurants and donors with NGOs and needy people.

## 🚀 Features
- AI Chef using Google Gemini API
- Food Donation system (Donor → Receiver/NGO)
- Customer ordering system
- Role selection (Customer, Donor, Receiver)
- Modern UI with animations

## 🧠 Google Services Used
- Google Gemini API for AI-based suggestions

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Purpose
To reduce food wastage and help distribute leftover food efficiently.
